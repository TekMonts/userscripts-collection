# 🚀 Userscripts Collection

A curated collection of powerful **Tampermonkey userscripts** designed for automation, debugging, and enhancing modern web applications.

Built for developers, power users, and anyone who wants more control over their browsing experience.

---

## ✨ Highlights

* ⚡ Lightweight, high-performance scripts
* 🧠 Focused on **automation & reverse engineering**
* 🌐 Works across multiple websites
* 🛠️ Developer-oriented utilities
* 🔓 Open-source & customizable

---

## 📦 Available Scripts

### 🔹 XHR/Fetch Logger Pro

Intercept and log all network requests made via `XMLHttpRequest` and `fetch`.

**Features:**

* Logs request & response data
* CSP bypass using `unsafeWindow`
* Works at `document-start`

**Install:**

[👉 Install Script](https://raw.githubusercontent.com/TekMonts/userscripts-collection/main/scripts/XHR-Fetch-Logger-Pro.user.js)

*(More scripts coming soon…)*
